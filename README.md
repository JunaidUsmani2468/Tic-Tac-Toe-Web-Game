# üéÆ Tic-Tac-Toe Web Game

A modern, interactive version of the classic Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This project features a clean UI, real-time score tracking, and dynamic turn logic.

![Game Screenshot](/assets/Tic-Tac-Toe-SS1.png)

## ‚ú® Features

- **Interactive Gameplay:** Smooth 3x3 grid for two players (X and O).
- **Smart Turn Logic:**
  - The game always starts with **Player O**.
  - **Winner Starts:** To keep things fair, the winner of the previous match gets the first turn in the next round!
- **Score Tracking:** Keeps track of wins for both Player X and Player O.
- **Match Statistics:** Tracks Total Matches Played and Total Draws.
- **Winner Announcement:** A custom popup modal announces the winner or if the game is a draw.
- **Responsive Design:** Beautiful Purple-themed UI that works on different screen sizes.

## üöÄ How to Play

1.  Clone the repository
2.  Open `index.html` in your web browser.
3.  **Round 1:** Player **O** makes the first move.
4.  **Subsequent Rounds:** The winner of the last game gets the advantage of moving first!
5.  Try to align 3 symbols horizontally, vertically, or diagonally to win.
